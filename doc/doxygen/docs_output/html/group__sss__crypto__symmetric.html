Symmetric cipher update.

Input data does not have to be a multiple of block size. Subsequent calls to this function are possible. Unless one or more calls of this function have supplied sufficient input data, no output is generated. The cipher operation is finalized with a call to sss_cipher_finish().

Parameters
[in]contextPointer to symmetric crypto context.
[in]srcDataBuffer containing the input data.
[in]srcLenLength of the input data in bytes.
[out]destDataBuffer containing the output data.
[in,out]destLenLength of the output data in bytes. Buffer length on entry, reflects actual output size on return.
Returns
Status of the operation
Return values
kStatus_SSS_SuccessThe operation has completed successfully.
kStatus_SSS_FailThe operation has failed.
kStatus_SSS_InvalidArgumentOne of the arguments is invalid for the function to execute.
